# chat-service
Service for chatting room

Build Process:
---------------
Clone project to folder
Open terminal in project root folder
Build project: mvn clean install <br>
Run project: mvn spring-boot:run  <br>

Tech Stack:
-----------
Java 17 <br>
Spring 3.X <br>
H2 In Memory DB <br>
Maven <br>
